# **App Name**: NEP Digital Platform

## Core Features:

- Multilingual Public Website: Display party information, news, and manifesto in multiple languages (EN, TA, HI, TE, ML, KN).
- Membership Registration: Allow ex-servicemen and supporters to register as members through a mobile number verified, multi-step form, and digital membership card generation.
- Donation System: Enable donations with UPI/QR/online payments, manual/auto verification, and receipt generation. Allow donations with or without membership.
- Admin Panel: Provide a secure admin panel with email-based login for member approval, payment verification, report generation (state, district, constituency), and data export.
- SEO Optimization: Optimize website content and structure for search engines, using semantic HTML, meta tags, and keyword research (National Ex Servicemen Party, Ex-servicemen political party India).
- Membership Card Design: AI powered feature which provides ability for members to design custom party ID membership card according to the theme.
- AI news summary: Summarize political news with a tool that chooses whether the full article should be rendered based on whether there's sufficient information

## Style Guidelines:

- Primary color: Sky Blue (#39A9DC) reflects the party's national identity.
- Secondary color: Olive Green (#6B7F2A) from the party flag.
- Background color: White (#FFFFFF) provides a clean backdrop.
- Accent color: Black (#1A1A1A) provides contrast and highlights important elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, offering both modern appeal and readability.
- Use clean, modern icons to represent various sections and actions on the website.
- Implement a clear and intuitive layout, prioritizing mobile-first design for accessibility across devices.
- Incorporate subtle animations and transitions to enhance user experience and engagement.
